# Obat-obatan
# Krim atau pil untuk mengendalikan infeksi. Untuk infeksi ringan, dokter mungkin akan meresepkan krim, losion, atau gel antibiotik. 
# Krim, sampo atau pil untuk melawan infeksi jamur. Obat-obatan ini diresepkan apabila folikulitis disebabkan oleh infeksi jamur. 
# Krim atau pil untuk mengurangi peradangan. Pengobatan ini sering ditujukan untuk pengidap folikulitis eosinofilik ringan. Dokter kemungkinan akan meresepkan krim steroid untuk mengurangi rasa gatal.
